# [Scenario] 마이링크 사용자 시나리오

본 문서는 마이링크 서비스를 이용하는 두 가지 주요 페르소나(방문자, 소유자)의 행동 흐름을 정의합니다. 모든 시나리오는 "**사용자는 [목적]하기 위해 [행동]한다**"의 형식을 따릅니다.

## 1. 방문자 (Visitor) 시나리오

### 1.1 페이지 방문 및 탐색
- 사용자는 **특정 크리에이터의 통합 링크를 확인하기 위해** 크리에이터의 SNS 프로필에 기재된 `domain.com/[displayName]` 주소로 접속한다.
- 사용자는 **소유자가 누구인지 파악하기 위해** 상단의 프로필 이미지(또는 닉네임)와 소개글(Bio)을 읽는다.
- 사용자는 **전체적인 링크 목록을 훑어보기 위해** 화면을 위아래로 스크롤한다.

### 1.2 링크 클릭 및 이동
- 사용자는 **원하는 콘텐츠로 이동하기 위해** 목록의 링크 아이템을 클릭한다.
- 사용자는 **현재 보던 페이지를 유지하면서 링크를 확인하기 위해** 클릭 시 새 탭으로 열리는 외부 사이트를 탐색한다.

### 1.3 서비스 유입
- 사용자는 **나만의 마이링크 페이지를 만들고 싶어** 하단에 노출된 "나만의 마이링크 만들기" 버튼을 클릭하여 메인 페이지로 이동한다.

---

## 2. 소유자 (Owner) 시나리오

### 2.1 가입 및 프로필 설정
- 사용자는 **서비스를 시작하기 위해** 메인 페이지에서 "구글로 시작하기" 버튼을 눌러 로그인한다.
- 사용자는 **나만의 고유 주소를 확보하기 위해** 지메일 아이디로 자동 설정된 `displayName`을 확인하고 필요 시 수정한다.
- 사용자는 **나를 소개하기 위해** 프로필 설정 메뉴에서 한 줄 소개(Bio)를 입력하고 저장한다.

### 2.2 링크 추가
- 사용자는 **공유하고 싶은 새 사이트가 생겨** "링크 추가" 버튼을 클릭한다.
- 사용자는 **아이콘을 별도로 준비하는 번거로움을 덜기 위해** URL만 입력하여 자동으로 생성되는 파비콘을 확인한다.

### 2.3 링크 수정 (Inline Edit)
- 사용자는 **링크의 제목을 더 매력적으로 바꾸기 위해** 목록에 노출된 타이틀 텍스트를 직접 클릭하여 수정 모드로 진입한다.
- 사용자는 **수정 사항을 빠르게 반영하기 위해** 텍스트를 고친 후 Enter 키를 누르거나 다른 곳을 클릭하여 즉시 저장한다.

### 2.4 링크 삭제
- 사용자는 **더 이상 유효하지 않은 링크를 제거하기 위해** 링크 아이템 우측의 삭제 버튼을 클릭하여 목록에서 지운다.

### 2.5 관리 및 확인
- 사용자는 **방문자에게 보여지는 최종 화면을 확인하기 위해** 자신의 고유 URL(`domain.com/[displayName]`)로 직접 접속하여 레이아웃을 검토한다.
- 사용자는 **어떤 링크가 인기 있는지 확인하기 위해** 관리 페이지의 각 링크 옆에 표시된 누적 클릭 수를 확인한다. (Phase 2)
